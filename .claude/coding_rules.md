# Coding Rules

- Use TypeScript wherever possible.
- Prefer explicit types over implicit any.
- Keep components small and composable.
- Keep business logic out of UI components.
- Use server actions/API routes only where appropriate.
- Write schema and validation for important inputs.
- Avoid premature optimization.
- Prioritize maintainability.

## AI Output Handling

AI-generated flavor vectors must be saved as review drafts first.
Never publish AI output directly without human verification.
