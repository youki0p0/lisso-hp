# LISSO Design Principles

Design the entire application autonomously.

Do not ask for colors, spacing, or layout unless absolutely necessary.

## Keywords

- Minimal
- Spacious
- Elegant
- Quiet
- Theoretical
- High-quality
- Gastronomy
- Infrastructure
- Security
- Precision

## Visual Direction

- Dark mode by default
- Black, white, and antique gold accents
- Typography-first
- Large margins and calm hierarchy
- Avoid excessive cards and borders
- Use motion sparingly
- Mobile first

## References

- Apple
- Linear
- Anthropic
- Stripe
- Notion

## Avoid

- Loud gradients
- Too many icons
- Cheap luxury look
- Overuse of gold
- Dense dashboard clutter
- Generic startup SaaS feel
