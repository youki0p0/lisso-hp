# Tech Stack

## Frontend

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui if useful
- Vercel hosting

## Backend

- Google Cloud Run
- Node.js or Python API
- REST or tRPC-style API
- AI provider abstraction layer

## Database

- Google Cloud SQL PostgreSQL
- Prisma ORM

## Storage

- Google Cloud Storage
- Recipe images
- OGP images
- Uploaded assets

## Auth

Recommended options:

- Firebase Auth for simple social/email login
- NextAuth for Next.js-native control

## Integrations

- X sharing via URL intent and OGP
- EC product links
- AI APIs
- Future: CSV import/export
