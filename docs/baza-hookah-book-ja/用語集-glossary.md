# 翻訳ガイド・専門用語集（БАЗА / シーシャ専門書）

## 翻訳方針（厳守）
- 原文（ロシア語・OCR由来）を **完全かつ忠実に** 日本語へ翻訳する。要約・省略・意訳での圧縮は禁止。一文一文すべて訳す。
- 段落構造を保持する。原文の段落区切りに対応させる。
- 文体: です・ます調を基本とし、インタビューの発言は自然な口語で。専門書として読めるように。
- OCRノイズの扱い: 単独のページ番号（例「25」）、柱・ヘッダー（例「БАЗА \ 02」「02\ БАЗА」「ГЛАВА 02 ГЛАВ」）、判読不能な文字列断片（例「POINT ART театор) дна и дос NHEN」「Λ」「>」単独）は実本文ではないので訳出しない。ただし**意味のある文・見出し・固有名詞は決して省略しない**。判断に迷う場合は訳出して [原文ママ: ...] と注記。
- 数値・単位・ブランド名・人名はそのまま（人名はカタカナ併記可）。ラテン文字のブランド名（Savinov Says, waterpipe.pro 等）は原綴を保持。
- 見出しは Markdown の見出し（##, ###）として整形してよい。
